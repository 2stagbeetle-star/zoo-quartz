## 背景・目的

冬季に活動量が低下していた動物たちは、春の気温上昇とともに代謝が高まり、採食・探索・社会的行動が活発化する。この時期に適切な**環境エンリッチメント**（以下、エンリッチメント）を提供することは、動物の行動欲求を満たし、常同行動の抑制や繁殖準備のコンディション維持に直結する。

冬眠や低活動期から「切り替わる」春季は、刺激の種類と強度を段階的に上げていく絶好のタイミングでもある。急激な刺激は逆にストレスになる場合があるため、本ガイドでは段階的な導入手順を解説する。

## 必要な準備・器材

| 器材・素材 | 用途 | 入手先 |
|-----------|------|--------|
| 麻袋・紙袋 | 採食エンリッチメントの包材 | 園内備品 |
| 松ぼっくり・枯れ枝 | 感覚刺激（嗅覚・触覚） | 園内採取（農薬無使用エリア限定） |
| 動物の糞・尿（他個体・他種） | 嗅覚エンリッチメント | 園内採取 |
| ゴム製おもちゃ・ボール | 操作エンリッチメント | 市販品 |
| スパイスや香草（無害なもの） | 嗅覚刺激 | 食品店 |
| 氷ブロック（フルーツ・肉入り） | 採食＋温度刺激 | 園内製作 |
| 点滴スタンド・ロープ | 吊り下げエンリッチメントの設置 | 園内備品 |

**事前確認事項:**
- 使用する素材に農薬・防腐剤・塗料などが付着していないか確認
- 当該動物の[[アレルギー・過敏症記録]]を参照
- 複数個体飼育の場合、採食エンリッチメントは個体数分以上を用意（競合・ストレス防止）

## 手順

### Phase 1: 冬季からの移行（3月上旬）

1. **現状の行動観察（3〜5日間）**
   - 活動時間帯・休息場所・移動ルートを記録
   - 常同行動（往復歩き・毛引きなど）の頻度を確認
   - 記録を[[行動観察日誌]]に残す

2. **嗅覚エンリッチメントから開始**
   - 展示場の隅に少量のスパイス（シナモン・ターメリックなど）を配置
   - 他個体・他種の糞を遠方にひと塊配置
   - 反応を観察し、過剰反応（逃避・攻撃）がなければ継続

### Phase 2: 採食エンリッチメントの導入（3月中旬）

3. **包材を使った採食**
   - 通常の給餌量の一部（20〜30%）を麻袋・紙袋に包んで配置
   - 慣れてきたら包み方を複雑にする（輪ゴム・テープで封をするなど）
   - 肉食獣には冷凍肉を氷ブロックに封入するのも有効

4. **分散配置**
   - 1箇所にまとめて置いていたエサを、展示場内3〜5カ所に分散
   - 採食時間の延長と歩行距離の増加を促す

### Phase 3: 操作・社会的エンリッチメントの追加（3月下旬〜4月）

5. **操作エンリッチメントの導入**
   - ロープに食材を結びつけた「ぶら下げ型エサ」を設置
   - ゴムボールや硬質プラスチックの容器にエサを詰めた「パズルフィーダー」を導入
   - [[パズルフィーダーの作り方]]を参照

6. **繁殖期を見据えた個体間関係の把握**
   - オス・メスの接近行動・誇示行動の頻度を観察
   - 攻撃的な相互作用が増えている場合はエンリッチメントで注意分散を図る

## 注意点・安全管理

- **素材の安全確認は必須。** 園外から持ち込む自然素材（枝・葉・土）は、農薬や重金属汚染のリスクがある。必ず採取エリアを管理し、獣医師に確認してから使用すること。
- **個体差を観察する。** エンリッチメントを嫌がる・無視する個体に無理に提供しない。反応がない場合は種類や提供方法を変える。
- **記録を必ず残す。** 「提供日時・種類・反応・継続/中止」を[[エンリッチメント記録台帳]]に記入。記録なきエンリッチメントは評価できない。
- **氷ブロックは溶解後の床面滑りに注意。** 特に高齢個体・骨格疾患のある個体の展示場では用いない。

## 事例・実績

**事例1: アムールトラ（雌5歳）への春季導入**
3月上旬から嗅覚→採食→操作の順に段階導入。常同行動（フェンス沿いの往復）が1日あたり平均45分→12分に減少。採食エンリッチメント提供日は展示場内の歩行距離が通常比で約2.3倍に増加（園内計測）。

**事例2: チンパンジー群れ（4個体）へのパズルフィーダー導入**
個体数+1個（5個）のフィーダーを分散配置したところ、採食をめぐる争いが前週比で70%減少。若齢個体（3歳）が最も長時間使用し、操作スキルの向上が観察された。
