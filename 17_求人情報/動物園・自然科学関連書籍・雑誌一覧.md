<div class="book-card" data-category="動物園・飼育員" data-keywords="旭山 小菅 革命 動物園 再建 行動展示">

### 📗 〈旭山動物園〉革命 ―夢を実現した復活プロジェクト

| 項目 | 内容 |
|------|------|
| **著者** | 小菅 正夫（元旭山動物園園長） |
| **出版社** | KADOKAWA（角川oneテーマ21） |
| **ISBN** | 978-4-04-710037-4 |
| **発行年** | 2006年 |
| **対象読者** | 動物園関係者・一般 |

**内容要約:**
廃園寸前まで追い込まれた旭川市立旭山動物園が、なぜ日本一の動物園に変貌できたのか。元園長・小菅正夫が初めて書き下ろした舞台裏。「行動展示」という革新的なコンセプトの誕生経緯、スタッフ全員が手描きした動物解説ポップ、予算ゼロから始まった再建の苦闘を詳細に語る。動物園の社会的役割や「命を伝える」という使命についての深い考察も含む。

**複数の情報源からの考察:**
上野動物園を抜いて日本一になった事実は、施設の豪華さではなく「見せ方の哲学」が集客を左右することを証明。WAZA（世界動物園水族館協会）の種保存重視の流れとも合致し、行動展示は現在の動物福祉研究（Animal Behaviour Science）でも支持される手法となった。

function filterBooks() {
  var query = document.getElementById('bookSearchInput').value.toLowerCase();
  var cat   = document.getElementById('bookCatFilter').value;
  var cards = document.querySelectorAll('.book-card');
  var count = 0;
  cards.forEach(function(card) {
    var text    = (card.dataset.keywords + ' ' + card.innerText).toLowerCase();
    var cardCat = card.dataset.category || '';
    var matchQ  = query === '' || text.includes(query);
    var matchC  = cat   === '' || cardCat === cat;
    if (matchQ && matchC) { card.style.display = ''; count++; }
    else                  { card.style.display = 'none'; }
  });
  var total = cards.length;
  document.getElementById('searchResultCount').innerText =
    (query || cat) ? count + ' 件表示中（全 ' + total + ' 件）' : '';
}
function resetFilter() {
  document.getElementById('bookSearchInput').value = '';
  document.getElementById('bookCatFilter').value = '';
  filterBooks();
}
