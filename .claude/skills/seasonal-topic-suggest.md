# 季節トピック提案スキル

## 概要
現在の月に基づいて、次に書くべき季節トピック記事の候補を3〜5件提案する。
既存の `02_季節トピック/` 内記事との重複チェックつき。

## 使い方
```
/seasonal-topic-suggest
# Vault のルートディレクトリで実行
```

## 実行手順

### ステップ 1: 現在の季節を判定

```bash
MONTH=$(date +%m)
```

| 月 | 季節 | 動物園のイベント・特徴 |
|---|------|---------------------|
| 3〜5月 | 春 | 繁殖シーズン開始、冬眠明け、新生動物 |
| 6〜8月 | 夏 | 暑熱対策、夜の動物園、水辺の生き物 |
| 9〜11月 | 秋 | 食欲増進、渡り鳥、冬眠準備 |
| 12〜2月 | 冬 | 寒冷対策、冬毛・換毛、温帯・熱帯種の管理 |

### ステップ 2: 既存記事のスキャン

`02_季節トピック/` 内の全記事タイトルを収集し、重複を避ける:
- 既存記事の `title:` フィールドを抽出
- 類似タイトルがすでにある場合は候補から除外

### ステップ 3: 候補の提案

現在の季節と「次に来る季節」の両方を考慮し、3〜5件の具体的なトピック候補を提案する。

**提案フォーマット（例: 3月の場合）:**

```
季節: 春（3月）

提案トピック:

1. **「繁殖シーズン到来！動物園の春の管理ポイント」**
   - タグ: #季節トピック #春 #繁殖
   - 概要: 春季の繁殖行動の観察・管理・展示対応

2. **「冬眠明けの動物たち――覚醒後の健康チェック手順」**
   - タグ: #季節トピック #春 #健康管理
   - 概要: ヒグマ・タヌキなど冬眠種の目覚め後の観察と体調管理

3. **「春の換毛期に知っておきたいグルーミング対応」**
   - タグ: #季節トピック #春 #健康管理
   - 概要: 換毛ストレスの見極めと適切なグルーミング補助

4. **「来園者が増える春、動物展示の見せ方を工夫する」**
   - タグ: #季節トピック #春 #展示
   - 概要: 春のハイシーズンに向けた展示説明・解説板の工夫

5. **「春の昆虫食動物への給餌変化――旬の生き餌を活用する」**
   - タグ: #季節トピック #春 #給餌
   - 概要: カエル・トカゲ・鳥類向けの春季給餌バリエーション
```

### ステップ 4: 提案の保存

`_drafts/季節トピック提案_YYYY-MM.md` として保存する。

## 完了確認
- 保存したファイルパスを表示する
- 「記事を生成する場合は `/weekly-article-generator` を実行してください」と案内する
