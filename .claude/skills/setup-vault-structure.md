# Vault フォルダ構造セットアップスキル

## 概要
Zoo Knowledge Vault の Obsidian フォルダ構造を初期構築する。
既存のフォルダを上書きせず、不足しているフォルダのみ作成する。

## 実行手順

1. 以下のフォルダ構造をカレントディレクトリに作成する:

```
00_Index/
01_飼育日誌/
  哺乳類/  鳥類/  爬虫類/  両生類/  その他/
02_季節トピック/
  春/  夏/  秋/  冬/
03_飼育ノウハウ/
  給餌管理/  健康管理/  環境エンリッチメント/  繁殖管理/  展示設計/
04_生態系ニュース/
  保全活動/  気候変動影響/  外来種問題/  法制度/
05_種別図鑑/
  哺乳類/  鳥類/  爬虫類/  両生類/
06_自然環境/
  生息地/  生態系/  フィールドワーク/
_templates/
_images/
_drafts/
.obsidian/
.claude/skills/
```

2. `00_Index/index.md` が存在しない場合のみ作成する。
   - タイトル: "Zoo Knowledge Vault"
   - カテゴリ一覧テーブル（各フォルダへのウィキリンク付き）
   - タグ体系の説明

3. `.obsidian/app.json` を作成（新しいリンクの形式: ウィキリンク、添付ファイル保存先: _images）
4. `.obsidian/templates.json` を作成（テンプレートフォルダ: _templates）

## 完了確認
- フォルダ一覧を表示して構造を確認する
- `00_Index/index.md` の内容を表示する
