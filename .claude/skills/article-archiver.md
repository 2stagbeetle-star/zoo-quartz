# 記事アーカイブ・自動分類スキル

## 概要
`_drafts/` フォルダ内にある完成した記事（`draft: false` または指定されたファイル）を、フロントマターのタグや内容に基づいて、適切なカテゴリフォルダ（`01_飼育日誌` 〜 `18_学術論文・研究`）に自動的に移動する。

## 実行手順

### 1. 対象ファイルの特定
`_drafts/` 内のサブフォルダをスキャンし、`.md` ファイルをリストアップする。
特に `draft: false` になっているものは「完成」とみなし、アーカイブ対象とする。

### 2. 分類ロジック
記事の内容またはタグ（`tags:`）をスキャンし、以下の優先順位で格納先を決定する。

| カテゴリ | 判定キーワード / タグ | 格納先フォルダ |
| :--- | :--- | :--- |
| **学術論文・研究** | `#研究`, `#論文`, `一次資料`, `DOI`, `発表`, `学術` | `18_学術論文・研究/` |
| **フィールドワーク** | `#フィールドワーク`, `#調査`, `#観察`, `#標本`, `#採集` | `16_フィールドワーク・調査手法/` |
| **獣医学・医療** | `#獣医学`, `#医療`, `#手術`, `#疾病`, `#治療` | `09_獣医学・動物医療/` |
| **保全・保護** | `#保全`, `#保護`, `#絶滅危惧`, `#野生復帰` | `08_動物保護・保全生物学/` |
| **水族館・水生生物** | `#水族館`, `#水生`, `#魚類`, `#サンゴ`, `#海洋` | `07_水族館・水生生物/` |
| **気候・環境問題** | `#気候変動`, `#温暖化`, `#プラスチック`, `#環境汚染` | `14_環境問題・気候変動/` |
| **生物分類図鑑** | `#生物分類`, `#分類学`, `#taxonomy` | `13_生物分類図鑑/` |
| **植物学・植生** | `#植物`, `#植生`, `#樹木`, `#草花`, `#植物学` | `12_植物学・植生/` |
| **生態学** | `#生態学`, `#個体群`, `#群集`, `#食物連鎖`, `#生態系` | `11_生態学・自然環境学/` |
| **生物学・生命科学** | `#生物学`, `#遺伝子`, `#進化`, `#DNA`, `#生命科学` | `10_生物学・生命科学/` |
| **自然環境** | `#自然`, `#国立公園`, `#森林`, `#湿地`, `#山岳` | `06_自然環境/` |
| **種別図鑑** | `#種別`, `#図鑑`, `#哺乳類`, `#鳥類`, `#爬虫類`, `#両生類`, `#昆虫` | `05_種別図鑑/` |
| **飼育ノウハウ** | `#給餌`, `#健康管理`, `#エンリッチメント`, `#繁殖` | `03_飼育ノウハウ/` |
| **季節トピック** | `#春`, `#夏`, `#秋`, `#冬`, `#季節` | `02_季節トピック/` |
| **運営・経営** | `#運営`, `#経営`, `#法制度`, `#JAZA`, `#広報` | `15_動物園学・博物館学/` |
| **求人情報** | `#求人`, `#採用`, `#募集` | `17_求人情報/` |
| **飼育日誌** | 上記に該当しない一般的な園館ニュース | `01_飼育日誌/` |

**【サブフォルダの選定】**
動物種タグがある場合は、各フォルダ内のサブフォルダへ移動する。
- 例: `#哺乳類` が含まれる保全記事 -> `08_動物保護・保全生物学/哺乳類/`
- サブフォルダが存在しない場合は、カテゴリ直下に配置する。

### 3. 移動の実行
- ファイルを移動する。
- 同名ファイルが移動先に存在する場合は、ファイル名に日付を付与する（例: `記事名_2026-03-31.md`）。
- `/link-integrity-check` を実行して、リンク切れがないか確認する。

## 完了報告
移動した記事のタイトル、タグ、および新しい保存先パスを一覧表で報告する。
