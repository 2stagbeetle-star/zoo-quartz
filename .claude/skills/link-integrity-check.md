# リンク整合性チェックスキル

## 概要
Vault 内の全 Markdown ファイルの `[[ウィキリンク]]` を収集し、対応するファイルが存在しない「孤立リンク」を検出する。
月次メンテナンスや記事追加後に実行する。

## 使い方
```
/link-integrity-check
# Vault のルートディレクトリで実行
```

## 実行手順

### ステップ 1: 全ウィキリンクの収集

以下のパターンで Vault 内の全 `.md` ファイルから `[[リンク名]]` を抽出する:
- `[[リンク名]]`
- `[[リンク名|表示テキスト]]`
- `![[埋め込み]]`

`_templates/`, `_drafts/`, `.obsidian/`, `.claude/` フォルダは除外する。

### ステップ 2: 対応ファイルの存在確認

抽出したリンク名ごとに、以下のルールで対応ファイルを探す:
1. `リンク名.md` がVault内のいずれかのフォルダに存在するか
2. `リンク名/index.md` が存在するか
3. いずれも存在しない → 「孤立リンク」として記録

### ステップ 3: レポート生成

`00_Index/リンクチェックレポート.md` に以下の形式で保存する:

```markdown
# リンク整合性チェックレポート

最終チェック日: YYYY-MM-DD

## サマリー

| 項目 | 件数 |
|------|------|
| チェックしたファイル数 | XX |
| 総ウィキリンク数 | XX |
| 孤立リンク（未作成記事） | XX |

## 孤立リンク一覧（新規記事候補）

これらのリンクは参照されているが、対応するファイルが存在しません。
新規記事作成の候補として活用してください。

| リンク名 | 参照元ファイル | 参照元カテゴリ |
|---------|-------------|--------------|
| [[ジャイアントパンダ]] | 05_種別図鑑/哺乳類/レッサーパンダ.md | 種別図鑑 |
| [[給餌スケジュール]] | 03_飼育ノウハウ/給餌管理/基礎知識.md | 飼育ノウハウ |

## 正常なリンク（参考）

問題のないリンクの一覧（折りたたみ）

<details>
<summary>正常リンク一覧（XX件）</summary>

...

</details>
```

## 完了確認
- レポートのパスを表示する
- 孤立リンク件数を表示する
- 孤立リンクが多い場合、上位5件の「新規記事候補」を提案する
