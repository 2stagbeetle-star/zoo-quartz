# 冬期フィールドワーク実践：積雪上の足跡調査で野生動物を記録する

## はじめに

野生動物の生息調査において、冬の積雪期は「見えない動物を見える化する」絶好の機会だ。葉が落ち、見通しが良くなった林床に雪が積もると、普段は見つけることが難しい中型・大型哺乳類の移動経路や行動が、足跡（トラック）としてくっきりと記録される。本記事では、初めてフィールドワークに挑む人向けに、冬期足跡調査の基本手順と記録方法を解説する。

## 足跡調査の基本

### なぜ冬の足跡調査が有効か

- **雪面は天然の記録媒体**：動物が通過した際の足跡形態・歩幅・歩行パターンがそのまま保存される
- **種の識別が容易**：足形の形状、爪痕の有無、足跡間距離（ストライド）から種を特定できる
- **行動推定が可能**：歩行・小走り・疾走・ホッピングなどの歩容の違いから行動を推定できる
- **コスト効率が高い**：自動撮影カメラや発信機なしに、低コストで広範囲を調査できる

### 調査に必要な装備

| 品目 | 備考 |
|------|------|
| 防寒・防水ブーツ | 長時間雪上を歩くため必須 |
| フィールドノート | 耐水性推奨。足跡スケッチを記録 |
| 巻き尺・スケール定規 | 足形サイズと歩幅の計測に使用 |
| デジタルカメラ | スケール（定規・コイン）を置いて撮影 |
| GPS機器またはスマートフォン | 発見地点の座標を記録 |
| 地形図（1:25000） | 移動経路と地形の関係を把握 |

## 主要な野生動物の足跡識別

### ニホンジカ（*Cervus nippon*）

蹄の二股構造による二つの爪痕が特徴的。前足と後足がほぼ同サイズ（長さ5〜7cm）。歩行時には前後足が重なって着地することが多い（オーバーステッピング）。雪が深い場所ではラッセルした跡が残る。

### ニホンタヌキ（*Nyctereutes procyonoides viverrinus*）

4本指で爪痕がある。足形は丸みがあり、全長4〜5cm程度。左右の足跡が比較的近い位置につく。タヌキの「ためぐそ（共同排泄場）」は冬でも利用されることが多く、足跡と合わせて確認すると確実性が高まる。

### キツネ（*Vulpes vulpes*）

犬に似た4本指足跡だが、タヌキより細長く、爪が細くはっきりしている。直線的な歩行パターン（ダイレクト・レジスター：後足が前足跡に正確に重なる）が典型的。

### ノウサギ（*Lepus brachyurus*）

後ろ足が非常に大きく（長さ8〜12cm）、前後の足が逆転したY字配置の跳躍跡が特徴的。後肢を前に踏み出してから前肢が着地するため、跡を追う方向が直感と逆になりやすい。

### テン（*Martes melampus*）

イタチ科特有の小型5本指跡（爪あり）。縦に長い足形と、2足ずつ並んだバウンディングパターン。ストライド（跳躍間距離）は体長の2〜3倍に達することもある。

## 調査手順

### 1. 調査地の選定

対象動物の生息が期待される環境（林縁、沢沿い、雪上の踏み分け道など）を事前に地図で確認する。積雪が少なくとも5cm以上ある状態が足跡記録に適している。降雪直後の「新雪」は表面が美しく保存されるが、気温が上がると雪が解けて形状が崩れるため、朝の早い時間帯の調査が望ましい。

### 2. トランセクト調査の設定

任意の経路をランダムに歩くのではなく、決まった幅（例：歩行路の左右2mずつ）の帯状区画（トランセクト）を設定し、その範囲内の足跡を記録する。同じルートを繰り返し調査することで、季節変化や年間変動を比較できる。

### 3. 記録方法

足跡を発見したら以下の項目を記録する。

- **種（または推定種）**
- **足形サイズ**（長さ・幅をmm単位で計測）
- **ストライド・ステップ幅**
- **歩行パターン**（歩行、小走り、疾走、ホッピング等）
- **足跡数・連続性**（単独通過か群れか）
- **付随するサイン**（糞、毛、齧り跡、巣穴等）
- **GPS座標・地形特性**
- **写真**（スケール付きで全体と個別の足形を撮影）

### 4. 石膏型取り（任意）

状態のよい足跡は石膏を流し込んでキャスト（型取り）することで立体的な記録が残せる。水に対する石膏の比率（約100g:60mLが目安）を雪の寒さで調整し、固化時間（20〜30分）を待って慎重に取り出す。博物館標本や教育素材として活用できる。

## 安全管理と倫理的配慮

- **複数人での行動**：積雪山地での単独行は危難が生じた際に対応が困難になる。必ず2名以上で行動し、行動計画を第三者に伝えておく
- **クマ対策**：冬眠明けや休眠状態のツキノワグマ・ヒグマに偶発的に遭遇するリスクがある。クマ生息地での調査はクマ鈴の着用と情報収集を徹底する
- **動物への配慮**：過度な追跡は動物にエネルギー消費を強いる。食物の少ない冬季に野生動物を不必要に刺激することは倫理的に避けるべきである
- **土地所有者・管理機関への確認**：国立公園内や私有林では事前の許可申請が必要な場合がある

## 動物園飼育スタッフとの接点

足跡調査の知識は動物園の飼育業務とも接点がある。屋外放飼場の積雪時に動物の足跡を記録することで、飼育個体の夜間・早朝の行動経路や採食場所を非侵襲的に把握することができる。また、繁殖期の雄の活動域変化や、新しく搬入した個体が展示内のどの場所を使っているかを把握する簡便な方法として活用されている事例もある。
