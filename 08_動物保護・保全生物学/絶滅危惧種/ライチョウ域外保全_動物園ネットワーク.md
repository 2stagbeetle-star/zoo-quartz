## 概要

ライチョウ（*Lagopus muta japonica*）は国の特別天然記念物であり、中央アルプス個体群が一時絶滅したことで域外保全の重要性が広く認識された。2026年春、富山市ファミリーパークから恩賜上野動物園への個体移動が報告され、動物園ネットワークを活用した種保存の実践が注目されている。

---

## ライチョウの基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Lagopus muta japonica* |
| 分類 | キジ目 ライチョウ科 |
| 生息地 | 日本アルプス・立山連峰・木曽山脈 |
| 個体数 | 約2,000〜3,000羽（推定） |
| 保護区分 | 国指定特別天然記念物（1955年） |
| IUCNステータス | 軽度懸念（LC）※日本亜種は深刻な危機 |
| 主な脅威 | 気候変動・登山者増加・外来捕食者（キツネ）|

---

## 中央アルプスにおける絶滅と復活

中央アルプスでは1980年代以降にライチョウが絶滅した。2020年から環境省と長野県が連携し、南アルプスから卵・ヒナを移送して中央アルプスへの再導入を試みた。2022年には野外での自然繁殖が確認され、域外保全と野生復帰の成功例として国際的にも注目されている。

> **関連記事:** [[08_動物保護・保全生物学/野生復帰・再導入]]

## 動物園ネットワークの役割

### 施設間の個体移動

ライチョウの飼育下繁殖では、遺伝的多様性を維持するために施設間での個体交換が不可欠。JAZAの種保存委員会が主導し、以下の施設が保全プログラムに参加している。

| 施設 | 役割 |
|------|------|
| 富山市ファミリーパーク | 繁殖拠点・高地環境対応施設 |
| 恩賜上野動物園 | 展示・教育拠点 |
| 長野市茶臼山動物園 | 繁殖・研究協力 |
| 大町山岳博物館 | 高山生態系研究 |

2026年3月、富山市ファミリーパークから上野動物園への1羽の移動がこの連携の一環。

### 飼育技術の確立

ライチョウは高山性の鳥類で、低地環境での飼育は難度が高い。克服すべき課題は以下の通り。

- **温度管理**: 夏季でも20℃以下を維持する必要がある空調設備
- **採食**: 高山植物（コケモモ・チングルマ・タカネスミレ等）の安定供給
- **感染症対策**: 低地特有の病原体（寄生虫・コクシジウム等）への感受性が高い
- **繁殖誘導**: 日長・気温変化による繁殖行動のコントロール

---

## 気候変動との闘い

ライチョウが生息する高山帯は、地球温暖化によって生息適地が縮小している。標高2,500m以上の積雪期間が短くなることで、天敵（オオタカ・キツネ）との接触機会が増加するリスクがある。

**IPCC第6次評価報告書（2021）** では、現状のペースで温暖化が進むと2070年代までに日本の高山帯が激変し、ライチョウの生息可能域が50%以上縮小するとの推計もある。

## 来園者への教育効果

動物園でのライチョウ展示は、環境問題への関心喚起に大きな効果を持つ。

- 高山環境・気候変動の問題を身近に伝えるシンボル種
- 「日本の絶滅危惧種」を実際に見られる数少ない機会
- 保全活動への寄付・支援につながるストーリーテリング
