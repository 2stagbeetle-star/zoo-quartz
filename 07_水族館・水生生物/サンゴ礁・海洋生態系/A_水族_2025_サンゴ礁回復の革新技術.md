## 基本情報
- **対象**: サンゴ礁生態系（特に造礁サンゴ類）
- **関連機関**: CSIRO、サザンクロス大学、フロリダ水族館、NOAA
- **信頼度**: 複数査読済み論文および公式プロジェクトレポート
- **重要度**: [S] サンゴ礁保全を「手作業の移植」から「産業レベルの大量増殖」へ転換させる革新技術群。

## 概要
2025年、サンゴ礁保全技術は大きな飛躍を遂げた。オーストラリアのCSIROとサザンクロス大学が開発した「幼生シードボックス（Larval Seedbox）」は、サンゴの産卵後の幼生を水中の育成ボックス内に留め、従来手法の最大56倍の定着率を達成。一方、フロリダ水族館は3Dプリント技術で作られた生分解性の「コーラルディフェンダー」を開発し、移植直後のサンゴを捕食魚から守ることに成功した。これらの技術革新は、個人の手作業で年間数百本だったサンゴの移植を、ロボット化と工業プロセスで年間数万本レベルに引き上げる可能性を秘めている。

## 詳細

### 幼生シードボックス（Larval Seedbox）
| 項目 | 内容 |
|------|------|
| 開発 | CSIRO / サザンクロス大学（豪） |
| 原理 | サンゴ産卵時の浮遊幼生を海中ボックス内に保持し、着底・定着率を飛躍的に向上 |
| 成果 | 従来のブロードキャスト法の56倍の成功率 |
| 適用 | グレートバリアリーフの大規模回復プロジェクト |

### コーラルディフェンダー（Coral Defender）
| 項目 | 内容 |
|------|------|
| 開発 | フロリダ水族館 / FWC共同 |
| 原理 | 3Dプリントされた生分解性のケージ構造で、移植直後のサンゴフラグメントを魚の食害から保護 |
| 素材 | 海洋環境で数ヶ月かけて分解されるバイオプラスチック |
| 意義 | 初期の最も脆弱な期間の死亡率を大幅に削減 |

### 補助進化（Assisted Evolution）
- **マイクロバイオーム工学**: サンゴのストレス耐性を高める共生藻類（褐虫藻）とプロバイオティクス細菌を意図的にペアリング。
- **温度馴化**: 高水温環境でサンゴを事前に養殖し、将来の温暖化に耐える個体を育成してから野外に移植。
- **UZELA（水中プランクトン集光装置）**: プログラム可能なLED光列により動物プランクトンを集め、サンゴの栄養を強化する自律型装置。

## 飼育員・飼育スタッフ目線の注目点
- **水族館が保全の「製造拠点」に**: 陸上養殖施設のノウハウ（水質管理、照明制御、給餌自動化）がサンゴの大量増殖にも直結。
- **来館者教育のストーリー性**: 「この水槽のサンゴが、いつか海に帰って礁を再生する」というナラティブは、保全教育の強力なツールとなる。
