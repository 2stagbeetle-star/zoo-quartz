## 概要

淡水エイ（Potamotrygonidae科）やポリプテルス（Polypteridae科）、アラパイマ（Arapaima gigas）に代表される「古代魚」は、数千万〜数億年の進化史を持つ魅力的な淡水魚群であり、水族館での展示・飼育の対象として世界的に高い人気を誇る。2023年のNature掲載研究は南米原産の淡水エイがインドネシアで外来種として定着した事例を報告し、アクアリウム由来の生物多様性への影響が「保全のパラドックス」として議論されている。本記事では、これらの種の生物学的特性と飼育技術、繁殖管理の実務を整理する。

## 淡水エイ（Potamotrygonidae科）

### 分類と進化

淡水エイはサメと同じ軟骨魚綱に属し、約1億年前に海産のエイから分岐したとされる。南米の河川系に適応放散し、現在30種以上が記載されている。

| 特性 | 詳細 |
|------|------|
| 分類 | 軟骨魚綱・トビエイ目・ポタモトリゴン科 |
| 分布 | 南米（アマゾン川、オリノコ川、パラナ川水系） |
| 寿命 | 飼育下で10〜15年以上 |
| 繁殖形態 | 卵胎生（体内で胚が発育、仔魚を出産） |
| 特記事項 | 尾部に毒棘を持つ（飼育時の安全管理が必要） |

### 代表的な飼育種

- **モトロ（*Potamotrygon motoro*）**: 最も一般的な飼育種、オレンジの眼状斑
- **ブラックダイヤモンド（*P. leopoldi*）**: 黒地に白い水玉模様
- **パール（*P. jabuti*）**: 真珠状の斑紋パターン

### 飼育管理のポイント

- **水槽サイズ**: 最低でも幅150cm×奥行60cm以上の底面積を確保
- **底砂**: 細かい砂を薄く敷き、エイが埋まれる環境を提供
- **水温**: 26〜30℃
- **pH**: 6.0〜7.0（弱酸性〜中性）
- **給餌**: 生餌（エビ、魚）、冷凍餌、人工飼料への馴致
- **安全対策**: 毒棘による事故防止のための取り扱いプロトコル

### 繁殖技術

淡水エイの飼育下繁殖は技術的に確立されつつあり、専門ブリーダーにより多くの種で繁殖成功が報告されている。

- **性成熟**: オスは体盤幅25cm前後、メスは30cm前後で成熟
- **交尾行動**: オスがクラスパー（交接器）をメスの総排出腔に挿入
- **妊娠期間**: 種により3〜6か月
- **出産**: 1〜数匹の仔魚を出産（親と同じ形態で生まれる）

## ポリプテルス（Polypteridae科）

### 「生きた化石」としての位置づけ

ポリプテルスはアフリカ原産の条鰭魚類で、約3億年前の古生代に分岐した原始的な魚類グループに属する。

- **ガノイン鱗**: 硬いエナメル質の鱗で覆われた体表
- **肺呼吸**: 浮袋が原始的な肺として機能し、空気呼吸が可能
- **水上移動能力**: 胸鰭を用いた陸上での短距離移動が可能
- **外鰓**: 幼魚期に両生類に類似した外鰓を持つ

### 飼育の要点

| 項目 | 推奨条件 |
|------|---------|
| 水温 | 25〜28℃ |
| pH | 6.5〜7.5 |
| 水槽サイズ | 種により90cm〜180cm水槽 |
| 蓋 | 必須（脱走防止、空気呼吸のための空間確保） |
| 底砂 | 細かい砂または底砂なし |
| 餌 | 冷凍赤虫、キャット、小魚 |

### 繁殖

ポリプテルスの繁殖は水族館でも実績がある。

- **繁殖誘発**: 水温の低下と上昇（雨季の再現）、水換えの頻度増加
- **産卵**: オスがメスの臀鰭をカップ状にして卵を受け取り、水草等に付着
- **孵化**: 3〜4日で孵化、外鰓を持つ幼魚が出現
- **幼魚管理**: 微小な生餌（ブラインシュリンプ幼生）による初期給餌

## アラパイマ（Arapaima gigas）

### 世界最大級の淡水魚

アラパイマは南米アマゾン川流域に生息する世界最大級の淡水魚で、体長3m以上、体重200kg超に達する。

- **空気呼吸**: 義務的空気呼吸者であり、15〜20分ごとに水面で呼吸
- **鱗の特殊性**: ピラニアの攻撃にも耐える多層構造の鱗（バイオミメティクス研究の対象）
- **親による保護**: オスが口腔内で仔魚を保護する行動

### 水族館での管理

- **巨大水槽**: 体長に見合った大容量水槽（数千リットル以上）が必須
- **ジャンプ防止**: 水面から飛び出す習性があるため、堅牢な蓋の設置
- **水温**: 24〜29℃
- **給餌**: 大型の魚、エビ、人工飼料

### 保全のパラドックス

Nature（2023年）に掲載された研究は、南米原産の淡水エイ *Potamotrygon* 属がインドネシアで野外に定着した事例を報告した。アクアリウム由来の放流が原因とされ、「保全の対象である生物が別の生態系では侵略的外来種になる」という矛盾が指摘されている。水族館・アクアリウムには、飼育個体の適切な管理と放流防止の責任が求められる。
